# python 内存管理
1.  python中变量无须事先声明，也不需要指定类型，这是动态语言特性。
2.  变量只是一个标示符，指向一个对象，而这个对象被创建在内存的堆中。
3.  python编程中一般无须关心变量的存亡，一般也不用关心内存的管理。
4.  python使用引用计数记录所有对象的引用数。当对象引用数变为0时，他就可以被垃圾回收处理。
