# Python 解释器安装
根据自己的系统选择合适版本的python解释器 https://www.python.org/downloads/
注意： 一个系统环境可以安装多个版本的python解释器，因此在使用python解释器运行需要注意解释器的版本。建议使用虚拟环境管理器将不同的python解释器环境隔离开来，常见的虚拟环境管理器有virtualenv，conda等。
