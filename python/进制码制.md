# 进制
常见的进制有二进制、 八进制、十进制、 十六进制。应该重点掌握二进制、十六进制。
> 二进制 二进制是逢二进一 ，计算机使用的就是二进制数，使用0b表示一个数是二进制。

> 八进制  八进制是逢八进一 ，八进制中出现最高的数字就是七 ，使用0o表示一个数是八进制。

> 十进制 十进制是逢十进一，十进制是人们日常生活中使用的进制。

>  十六进制 十六进制是逢十六进一，一般使用0x表示一个十六进制数。
