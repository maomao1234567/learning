# 封装与解构
``` python
t = 1, 2
t = (1, 2) # 封装
x, y = (1, 2) # 解构
```
###  封装
封装指的是将单独的元素组合成一个线性的集合对象如元组，列表等。 python中交换两个变量的值就可以使用封装。
``` python
a = 1
b = 2
a, b = b, a # 这里先是b, a封装cheng
```
